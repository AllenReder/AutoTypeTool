# AutoTypeTool

## 中文简介

自动输入文本, 即使禁止粘贴也可以使用. 

## English Introduction

Automatically simulate input, even if pasting is disabled.

---

## How to use?

### Python

1. install all requirements
2. `python main.py`

Or you can use `autoType(text, interval, delay)` from `AutoTypeTool` package.
